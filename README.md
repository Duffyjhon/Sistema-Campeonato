# Sistema de Campeonato (Tabela de Pontos)

Este projeto é uma simulação de um campeonato esportivo, desenvolvida para fins educacionais, com foco em lógica de programação e organização de código.

O sistema controla partidas, pontuação e gera uma tabela de classificação com base nos resultados, permitindo visualizar o ranking dos participantes.

## Funcionalidades
- Cadastro de jogadores
- Simulação de partidas
- Cálculo automático de pontos
- Geração de tabela de classificação

## Tecnologias Utilizadas
- JavaScript
- React.js
- Firebase
- HTML e CSS

## Observações
Este projeto é uma simulação criada para fins de estudo.
Os nomes dos jogadores são genéricos e utilizados apenas para demonstração.
